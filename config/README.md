## 🎯 配置说明

### 📋 统一配置文件 (unified_config.py)
包含以下配置模块：
- **基础配置**: 股票池、保护区间等核心参数
- **波动性计算**: 算法参数和预设配置
- **性能优化**: API调用、缓存、界面更新等
- **talib集成**: 技术指标库参数
- **日志系统**: 日志级别和文件管理
- **开发配置**: 调试模式、测试配置等

### 🔧 使用方法
```python
# 导入统一配置
from Config.unified_config import *

# 或者按模块导入
from Config.unified_config import ADVANCED_CONFIG, TALIB_PARAMS
```
